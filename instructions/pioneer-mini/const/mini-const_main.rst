Сборка "Пионер Мини"
====================

Прочитав данную инструкцию вы сможете собрать "Пионер Мини"

.. important:: В наборе "Пионер мини" есть все элементы и инструменты для сборки и запуска квадрокоптера.

.. image:: /instructions/pioneer-mini/media/mini-sostav.PNG



Для сборки и настройки "Пионер мини" пройдите в следующие подразделы:

:doc:`mini-sborka`

:doc:`../settings/mini-settings_main`



В разделе :doc:`../../../programming/programming` подробно рассказано о том, как запрограммировать квадрокоптер.

.. note:: Программирование на базовом уровне, идентично программированию `стандартного Пионера`_ , поскольку они используют одинаковый автопилот.  Для работы с ESP-32 смотрите страницу "Работа с ESP32" (скоро будет)

.. _стандартного Пионера: ../../../instructions/pioneer-standart/main-standart.html


.. note:: Технические характеристики, внешний вид и комплектация товара могут быть изменены производителем без предварительного уведомления.
