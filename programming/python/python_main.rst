Последнее обновление ПО "Пионера Мини" позволяет программировать его на популярном языке Python. В данном разделе инструкции описано как настроить среду для программирования, описаны методы библиотеки pioneer_sdk, а также разобраны примеры кода из SDK:

*  Описание методов `pioneer_sdk`_

*  `Описание`_ работы скриптов

С примерами Python-скриптов вы можете ознакомиться в папке **examples** в `Github репозитории библиотеки pioneer_sdk <https://github.com/geoscan/pioneer_sdk>`__

О том, как совершенствовался подход к работе с языками программирования в линейке Пионеров и чему Вы сможете научиться на "Пионере мини" с помощью Python, читайте в нашей статье на `Habr <https://habr.com/ru/post/542960/>`_.

По вопросам и предложениям можете писать нам в телеграм канал `Геоскан Пионер <https://t.me/GeoscanPioneer>`__ или писать нам на почту **pioneer_sdk@geoscan.aero**
